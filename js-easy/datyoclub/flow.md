# ダチョウ倶楽部ゲーム

## 基本問題
- 「俺がやる!!!」ボタンを押す。
- 押した数を数える。
- 「どうぞどうぞぉ～」ボタンを押す
- 数えた数分、アラートで“どうぞどうぞ”を表示する。
- 数をリセットする。

```mermaid
flowchart TD
1[開始] --> 2[変数countに0をセット];
```

### 俺がやる!!!ボタンのクリックイベント
```mermaid
flowchart TD
1[俺がやる!!!ボタンを押す] --> 2[変数countを+1する];
```

### どうぞどうぞボタンのクリックイベント
```mermaid
flowchart TD
1[どうぞどうぞボタンを押す] --> 2[変数countの数だけどうぞどうぞを繰り返してアラート];
2 --> 3[変数countを0にする];
```
